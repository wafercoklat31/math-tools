# math-tools

ini adalah sebuah tool atau alat untuk membantu melakukan perhitungan matematika sederhana, disini untuk sekarang hanya bisa melakukan perhitungan luas bangun datar dan volume bangun ruang saja. dan untuk opsi nya ada:

luas :
1. luas persegi
2. luas persegi panjang
3. luas segitiga
4. luas lingkaran

volume:
1. volume kubus
2. volume balok
3. volume tabung
4. volume limas segi tiga
5. volume limas segi empat
6. volume limas segi lima
7. volume limas segi enam
8. volume prisma
9. volume kerucut
10. volume bola
